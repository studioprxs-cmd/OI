# OI 운영 정책 v1

## 1. 목적
OI 서비스의 이슈 등록, 검증, 모더레이션, 정산의 기본 원칙을 정의한다.

## 2. 운영 원칙
1. **투명성:** 이슈/결과/정산 근거를 공개한다.
2. **일관성:** 동일 유형 이슈에 동일한 판단 기준 적용.
3. **추적성:** 주요 변경 이력(누가, 언제, 무엇을) 기록.
4. **중립성:** 정치/상업적 이해관계로부터 독립적 운영 지향.

## 3. 이슈 등록 기준
- 검증 가능한 주장만 등록 가능
- 질문은 명확하고 단일 판단 가능해야 함
- 마감시각과 결과 판정 기준(정량/정성)을 사전 명시

## 4. 출처(Source) 기준
- 1차 출처 우선(공식 발표, 원문 데이터)
- 2차 기사 사용 시 원출처 병기 권장
- 클릭베이트/허위정보 매체는 제한 가능

## 5. 참여 정책
- 계정당 이슈별 투표 1회
- 베팅은 내부 포인트로만 진행
- 자기참조/다중계정/조작 행위 금지

## 6. 결과 확정(Resolution)
- 마감 후 운영자가 근거 검토 후 확정
- 상태: PENDING_REVIEW -> CONFIRMED / REJECTED
- 분쟁 발생 시 재검토 1회 가능, 변경 이력 공개

## 7. 정산 정책
- CONFIRMED 결과 기준 자동 정산
- 정산 내역은 WalletTransaction에 영구 기록
- 취소(CANCELED) 이슈는 베팅 원금 환급

## 8. 신고/제재
- 신고 유형: 스팸, 욕설/혐오, 허위정보, 저작권, 기타
- 처리 목표: 24시간 내 1차 조치
- 반복 위반자는 경고 -> 일시정지 -> 영구정지

## 9. 개인정보/보안
- 최소한의 사용자 정보만 보관
- 민감정보 저장 금지
- 관리자 권한 접근은 로그 기반 감시

## 10. 정책 변경
- 정책 변경 시 버전/시행일/변경 사유 공지
- 중대한 변경은 시행 24시간 전 사전 고지
